Module plugins

The plugins module provides abstraction / base classes for jQuery plugins and some constructor/destructor stuff. jQuery plugins can either be created extending the Plugin class or if the jQuery UI Widget Factory is preferred, using the Widget class. ```typescript export class MySuperWidget extends Widget {

} ```
